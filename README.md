# 🚀 GramSetu: Smart Village Super App

**GramSetu** is a comprehensive, full-stack rural governance and financial inclusion platform designed to digitize village administration and empower local communities.

---

## ✨ Key Features (प्रमुख वैशिष्ट्ये)

### 🏛️ Digital Governance (डिजिटल प्रशासन)
- **Notice Board**: Digital announcements with **AI Audio (Marathi Text-to-Speech)**.
- **Complaint Management**: Register complaints with **Smart AI Analysis** for urgency and categorization.
- **WhatsApp Integration**: Instant updates to citizens via WhatsApp.
- **Certificate Requests**: Online application for local certificates.
- **Polls & Voting**: Community decision-making through digital polls.

### 💰 Rural Fintech (ग्रामीण आर्थिक सक्षमीकरण)
- **Bachat Gat (SHG)**: Digital passbook, savings tracking, and loan management for women.
- **Community Finance (Vyaj System)**: Interest-based community lending system for men.
- **Online Payments**: Simulated **Razorpay Payment Gateway** for donations and savings.
- **Donation Transparency**: Live tracking of public funds and festival contributions.

### 🛍️ Local Economy (स्थानिक अर्थव्यवस्था)
- **Women's Marketplace**: E-commerce platform for local products (Papad, Pickles, etc.) with WhatsApp ordering.
- **Business Directory**: Local "Yellow Pages" for village shops and services (Electricians, Doctors).
- **Job Board**: Connecting local workers with employers for farm and retail jobs.

### 📊 Smart Analytics & Accessibility
- **Admin Insights**: Data visualization using **Chart.js** for village progress tracking.
- **Weather Integration**: Real-time weather alerts for farmers.
- **Dark Mode**: Premium user experience with persistent theme switching.
- **Report Generation**: One-click printable village development reports.

---

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Vanilla), Bootstrap 5, JavaScript, Chart.js.
- **Backend**: Core PHP, MySQL (PDO).
- **Security**: Role-Based Access Control (RBAC), Prepared Statements.
- **Integrations**: WhatsApp API (URI), Razorpay (Simulated), Web Speech API.

## 🚀 Installation
1. Clone the repository: `git clone https://github.com/luckyyy08/GramSetu.git`
2. Import `database.sql` into your MySQL server.
3. Update `config/db.php` with your credentials.
4. Run on XAMPP/WAMP: `http://localhost/GramSetu/`
